# SHOPPINGI - Plateforme E-commerce Intelligente

## 📋 Aperçu du Projet

**SHOPPINGI** est une plateforme e-commerce intelligente développée dans le cadre d'un projet de fin d'études en Génie Logiciel & Intelligence Artificielle. Cette solution révolutionnaire intègre des technologies d'IA avancées pour optimiser l'expérience utilisateur et la gestion commerciale.
## 🚀 Video Demo

🔗 **[Live Demo](https://drive.google.com/file/d/1LWdUmnJlevLbTkHHUrqGEzh-0WDdc2_l/view?usp=sharing)** 
## 🎯 Objectifs

- Créer une plateforme e-commerce avec des fonctionnalités d'IA avancées
- Développer un système de recommandation personnalisé
- Intégrer l'analyse des sentiments des avis clients
- Implémenter des outils prédictifs pour la gestion des ventes
- Offrir une expérience utilisateur optimisée et intuitive

## 🚀 Fonctionnalités Principales

### 🤖 Intelligence Artificielle
- **Système de recommandation** basé sur le filtrage collaboratif (SVD, NMF, KNN)
- **Filtrage basé sur le contenu** utilisant TF-IDF et clustering K-means
- **Analyse des sentiments** des avis clients avec LSTM et GloVe embeddings
- **Prédiction des ventes** utilisant ARIMA, SARIMA et Prophet

### 👥 Gestion Multi-Acteurs
- **Clients** : Navigation, commandes, avis, liste de souhaits
- **Vendeurs** : Gestion produits, statistiques, promotions
- **Administrateurs** : Supervision globale, gestion utilisateurs
- **Visiteurs** : Consultation libre des produits

### 💡 Fonctionnalités Métier
- Gestion complète du catalogue produits
- Système de paiement sécurisé avec Stripe
- Tableaux de bord analytiques
- Notifications intelligentes
- Gestion des promotions automatisée

## 🛠️ Architecture Technique

### Stack Technologique
- **Frontend** : React.js + TypeScript + Apex chart
- **Backend** : Node.js + Express.js
- **IA/ML** : Python + FastAPI
- **Base de données** : MongoDB
- **Paiements** : Stripe API

### Architecture
- **Pattern MVC** pour une séparation claire des responsabilités
- **API REST** pour la communication entre services
- **Microservices** pour les fonctionnalités IA
- **Architecture scalable** et modulaire


### Configuration
```bash
# Frontend (React)
npm install
npm start

# Backend (Node.js)
npm install
npm run dev

# Service IA (Python)
pip install -r requirements.txt
uvicorn main:app --reload
```
